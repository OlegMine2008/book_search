# Информационная программа о местоположении книг

**Функции**

1 - _Поиск книг по названию, идентификационному коду и по автору_
2 - _Ввод, редактирование и удаление информации о книгах в БД_
3 - _Просмотр информации о книге_
4 - _Ввод и просмотр личных заметок о книге_

**Дизайн**

Главное окно: Нередактируемая таблица, отоброжающая следующую информацию из таблицы в БД: код, название, 
автор, информация, заметки, в правой части - кнопки поиска, подключения БД, добавления книги, правки информации и заметок
Нередактируемая таблица: QTableView, имеющий в себе 5 столбцов
Кнопки: QPushButton, открывающие окна, запрашивающие информацию - папку БД, метод поиска(по коду, автору или названию),
информацию о новой книге(название, имя автора), изменение информации или заметок
Структура уточняющих окон: QLabel, QPushButton для подтверждения, QFileDialog для выбора БД или QLineEdit для редактирования
Заметок и Информации

**Архитектура кода**

- Библиотеки и данные о дизайне
- Класс, реализующий программу
    - Реализация дизайна
    - Назначение фнукций кнопкам
    - Функции кнопок и виджетов
- Главная функция, запускающая программу

**Структура Базы Данных**

Таблицы: 2
Структура таблицы books(в столбцах): 
Код integer 
Название text
Автор integer
Информация text
Заметки text

Структура таблицы authors: 
Код integer
Имя text
